
===========部署操作==============
mvn clean source:jar install
	项目打包同时生成源码包

===========注意事项==============
1.如果xml中有关"创建时间"的默认值，则修改xml和修改数据库(加上默认值)
2.如果有like语句,修改相应xml的语句加上 CONCAT('%',#{province},'%')
